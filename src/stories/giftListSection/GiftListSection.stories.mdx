# Gift list section

Basically this is a standalone section that shows gift list cards, you can place it anywhere
you need in the webpage.

As his main functions:

- Display information of a group of gift List cards in a responsive way
- Gives an action to all his gif list cards childs

Has media querys that let him adapt to any device screen
in desktop has an horizontal scroll to see all the elements
in tablets and mobile the elements stack in a list.
Has placeholder when there is no
